# Дипломный проект по профессии «Инженер по тестированию»  
Студент: Корецкий В.П.  

## Цели проекта  
Эта дипломная работа поможет закрепить знания и навыки по автоматизации тестирования мобильных приложений.
## В результате выполнения вы:
1. Проведёте ручное тестирование мобильного приложения «Мобильный хоспис».
2. Составите чек-лист для проверки приложения.
3. Распишите тест-кейсы для проверки приложения.
4. Автоматизируете проверку тест-кейсов по чек-листу.
5. Составите отчёты о тестировании.
## Описание приложения  
Приложение даёт функционал по работе с новостями хосписа и включает в себя:  
* информацию о новостях и функционал для работы с ними;
* тематические цитаты;
* информацию о приложении.

_Рис. 4. Функциональная схема мобильного приложения (Схема построена с помощью [сервиса](https://web.mindonmap.com "MindOnMap"), [ссылка на схему](https://web.mindonmap.com/view/8571d6eddd6ae3d9 "Cхема приложения"))_  
  
Работа с приложением начинается с авторизации, где необходимо ввести логин и пароль, нажать кнопку SIGN IN (Войти). При валидном вводе логина или пароля открывается основная страница приложения.  

### Элементы основной страницы:
1. Гамбургер меню (боковое левой меню). При нажатии открывается окно со следующими элементами:
* Главная;
* Новости;
* О приложении.
2. Элемент навигации - переход на страницу с цитатами (бабочка)
3. Элемент навигации - Logout
4. Раздел Новости, при нажатии которого выпадают элементы: с управляющими элементами DropDown  
* ВСЕ НОВОСТИ - линк на страницу с новостями;  
* только что добавленные новости;  
* Already editing news;  
* Test creating news.  
  
## Элементы страницы Новости:
1. Верхнее управляющее меню со следующими инструментами:  
* сортировка новостей;  
* фильтр новостей;  
* редактор новостей;  
2. Список новостей с управляющими элементами DropDown, в которых указывается название и дата новости.  
  
## Элементы страницы О приложении:
1. Содержит две гиперссылки на ВЕБ-ресурсы:  
* Политика конфиденциальности;  
* Пользовательское соглашение.  

## Раздел Главное меню -> Раздел с цитатами:
1. Блоки с цитатами с управляющими элементами DropDown, при нажатии на которые выпадают ответы на цитаты или ничего не выпадает.  
  
## Раздел Новости переход по элементу Фильтровать новости:
* блок Категории с управляющим элементом DropDown;  
* заполняемая форма Дата (начальная): реализован инструмент Data Picker;  
* заполняемая форма Дата (конечная): реализован инструмент Data Picker;  
* кнопка ФИЛЬТРОВАТЬ;  
* кнопка ОТМЕНА.  
  
## Раздел Новости переход в редактор новостей:  
1. панель управления с инструментами:  
* сортировка новостей;  
* фильтр новостей;  
* создание новости.  
2. блоки с новостями, содержащие следующие элементы:  
* название новости;  
* дата публикации;  
* дата создания;
* автор;
* статус (активна);
* меню управления с элементами:  
    * удалить новость;
    * редактировать новость;
    * управляющий элемент DropDown при нажатии на который выпадает информация о новости.
  
## Раздел Новости Редактор новостей переход в раздел Фильтровать новости:
1. блок Категории с управляющим элементом DropDown;
2. заполняемая форма Дата (начальная): реализован инструмент Data Picker;
3. заполняемая форма Дата (конечная): реализован инструмент Data Picker;
4. чекбокс Активна;
5. чекбокс Не активна;
6. кнопка ФИЛЬТРОВАТЬ;
7. кнопка ОТМЕНА.
  
## Раздел Новости Редактор новостей переход в раздел Создание новости:
1. блок Категории с управляющим элементом DropDown;
2. заполняемая форма Заголовок;
3. заполняемая форма Дата публикации: реализован инструмент Data Picker;
4. заполняемая форма Время: реализованы
* инструмент Time Picker;
* заполнение с клавиатуры (отдельной заполняется час и минуты);
5. заполняемая форма Описание;
6. переключатель-бегунок Активна;
7. кнопка СОХРАНИТЬ;
8. кнопка ОТМЕНА.

## Раздел Новости Редактор новостей переход в раздел Редактирование новости:
1. блок Категории с управляющим элементом DropDown;
2. заполняемая форма Заголовок;
3. заполняемая форма Дата публикации: реализован инструмент Data Picker;
4. заполняемая форма Время: реализованы
* инструмент Time Picker;
* заполнение с клавиатуры (отдельной заполняется час и минуты);
5. переключатель-бегунок Активна;
6. кнопка СОХРАНИТЬ;
7. кнопка ОТМЕНА.
